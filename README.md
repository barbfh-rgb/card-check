# Card Check

Card Check is a static offline first PWA for checking Swiss 1JJ Tarot card meanings after a reading is laid out.

## Deploy

1. Commit these files on the `main` branch.
2. In GitHub Pages, publish from the root of `main`.
3. Keep `start_url` and `scope` as `/` in `manifest.json` or the installed app will 404.

No build step is needed. Open `index.html` locally or serve the folder as a static site.
